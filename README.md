# Solucion Tarea1 StudyJam


Dentro de este repositorio se encuentra la tarea 1 de Study Jam, en el cual hacemos la copia de interfaz grafica
de una aplicacion movil.


## Requerimientos

  * Android Studio 2.3
  * Gradle 2.3
  * minSdkVersion 15

## Uso

Para probar este ejemplo clona este repositorio de la siguiente forma:
>
>     $ git clone https://github.com/rony60/StudyJam_Tarea1.git

Dentro de Android Studio:

* File --> New --> Import Project
* Seleccionas la ruta donde hiciste el `clone` del proyecto.
* Build --> Rebuild Project
* Run
